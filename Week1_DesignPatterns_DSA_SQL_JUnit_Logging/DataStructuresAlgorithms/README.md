# Week 1 - Data Structures & Algorithms

This folder contains the mandatory hands-on exercises for **Week 1 - Data Structures & Algorithms** as part of the Cognizant Java Full Stack Engineer (FSE) Skill Program.

---

## Exercises

### 1. E-commerce Platform Search Function

- Implemented **Linear Search** and **Binary Search** algorithms.
- Compared their time complexities.
- Analyzed which algorithm is more suitable for an e-commerce platform.

---

### 2. Financial Forecasting

- Implemented a recursive algorithm to predict future values.
- Analyzed the time complexity of the recursive solution.
- Discussed possible optimizations.
