# Financial Forecasting

## Problem Statement

Develop a financial forecasting tool that predicts the future value of an investment based on a fixed annual growth rate using a **recursive algorithm**.

---

### Project Structure

```text
FinancialForecasting/
├── FinancialForecast.java
├── ForecastDemo.java
└── README.md
```

---

### Output

```text
Current Value : ₹10000.0
Annual Growth Rate : 10.0%
Years : 5
Predicted Future Value : ₹16105.10

Time Complexity : O(n)
Space Complexity : O(n) (due to recursive call stack)
```
